Calculator

This Calculator Web-Application is a responsive, user-friendly tool built with HTML, CSS, and JavaScript that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

It features a modern interface with clickable buttons, keyboard input support, a clear display area, and functionality for clearing or deleting entries. The app includes a real-time calculation history panel that users can toggle or clear, and supports both light and dark themes via a mode switch.

The app features a classic and minimalist design with a sleek button layout and clean interface, offering a distraction-free and visually appealing user experience.
